Virtualization


Virtualization is a technique that is used by developers to run multiple operating systems in various virtual machines while running on the same exact host. This technique helped developers by reducing the required amount of hardware and resources that some projects might have needed. Virtualization also allows for easy recovery of data and maintenance in case errors occur in the process. Since virtualization reduces the amount of resources being used, it also reduces the cost of necessary hardware since there is less needed. There are some detriments to using this technique however. If there are too many virtual machines on a single host then it can lead to performance issues. There may also be a longer startup time due to the amount of virtual machines running at the same time on that single host.
